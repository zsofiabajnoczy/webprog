Bootstrap

A Bootstrap egy nyílt forráskódú keretrendszer (framework), mely HTML, CSS, JavaScript technológiákat használ.

A Bootstrap előnye, hogy a legtöbb „problémára”, mellyel egy fejlesztő találkozik weboldalak arculatának kialakításakor, már egy előre elkészített megoldást kínál, ami vagy teljesen megfelel az adott célra, vagy pedig minimális átalakítással.

Használata során előfordulhat, hogy bizonyos komponenseket mégis módosítani szeretnénk, így egyedivé téve az oldalt. Erre van lehetőség, ám nem ajánlott ez a gyakorlat, mivel a frissítések során visszaállnak a beállítások az alapállapotba. Ha mégis módosítani szeretnénk, akkor ezt érdemes egy különálló CSS fáljban megtenni, így ezt nem fogja érinteni a frissítés.

Dátum Verzió Történet
2011 Twitter Blueprint / Bootstrap Mark Otto és Jacob Thornton a Twitternél megunták a sok, különféle belső eszköz folyamatos összehangolását és elkezdtek egy keretrendszeren dolgozni. Ez volt leánykori nevén a Twitter Blueprint, amelyet a megjelenés előtt egyszerűen csak Bootstrap-nek neveztek, s amelyet végül 2011. augusztus 11 -én open source projekt formájában publikáltak.
2012 Bootstrap 2 A projekt hatalmas sikert aratott, sőt egy egsz közösség épült köré, ennek köszönhetően pedig folyamatosan fejlődni tudott. Évente adták ki a verziókat, míg végül 2014-ben megjelent a Bootstrap 4, amely több mint 6 évig élt.
2013 Bootstrap 3
2014 Bootstrap 4
2021 Bootstrap 5 2021. május 5-én egy újabb áttörés történt, ugyanis több hónapnyi fejlesztés után megjelent a várva várt Bootstrap 5.
